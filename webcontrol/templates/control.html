Solenoid valves — LN₂ manifold

“actual” is the energised state read back from the PLC; “desired” is the last command sent. Auto-open / auto-close arm the PLC ladder’s autofill enables for that vessel.

Heaters — PID loops

Everything is in Kelvin. Expression fields accept arithmetic over the aliases rtd1..rtd6, tc1..tc4 (each = the latest value_k in K) plus direct register refs DF<n> / DS<n> / C<n> — e.g. rtd1 + 5 (5 K above the LJ nozzle RTD). The expression is re-evaluated each poll; the K result is converted to °C only when written to the CLICK's DF (which stores temperatures in °C). Caveat: DF<n> returns the raw register contents — for the PID's temperature DFs that's °C, so add 273.15 if mixing with a Kelvin alias.
